## Who Can Contribute

Contributions are accepted **by invitation only**.

If you are interested in collaborating, contact:
👉 https://github.com/waren23greg-stack

Do NOT submit pull requests without prior written approval.
Unsolicited PRs will be closed without review.

---

## If You Are an Invited Collaborator

### What You May Do
- Fix bugs on assigned modules only
- Improve documentation with approval
- Write tests for existing functionality

### What You May NOT Do
- Modify core architecture (ORACLE / PHANTOM / COVENANT / BioGuard agents)
- Redistribute any portion of the code
- Use this code in any external project
- Remove or alter copyright headers

---

## Code Standards

- Python 3.10+
- PEP 8 formatting
- Every function must have a docstring
- Add copyright header to every new file:

```python
# EDEN-BioGuard — Copyright (c) 2026 Warren Greg. All Rights Reserved.
# Unauthorized use, copying, or distribution is strictly prohibited.
```

## Reporting Issues

Open a GitHub Issue with:
- Clear title
- Steps to reproduce
- Expected vs actual behaviour
- Your environment (OS, Python version)
